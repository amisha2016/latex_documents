Description

This web map references the live tiled map service from the OpenStreetMap (OSM) project. OpenStreetMap (OSM) is an open collaborative project to create a free editable map. for details move down of the world. Volunteers gather location data using GPS, local knowledge, and other free sources of information and upload it.




The resulting free map can be viewed and downloaded from the OpenStreetMap server. It is made available as a basemap for GIS work in ESRI products under a Creative Commons Attribution-ShareAlike license.

Surveys and personal knowledge

Map data is collected from scratch by volunteers performing systematic ground surveys using tools such as a handheld GPS unit, a notebook, digital camera, or a voice recorder. The data is then entered into the OpenStreetMap database.
